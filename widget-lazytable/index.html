You can set custom table header in <th> tag with 'data-table' attribute
or just let it use text.

HTML Markup

<table class="responsive-table">
    <thead>
        <tr>
          <th data-table="Custom Header 1"></th>
          <th></th>
          ...
        </tr>
    </thead>
    
    <tbody>
        <tr>
          <td></td>
          <td></td>
          ...
        </tr>
    </tbody>
</table>

jQuery

$('.responsive-table').lazyResponsiveTable();

*You still need to custom table style with CSS
